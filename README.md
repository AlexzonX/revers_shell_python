## 📜 Описание

Простая реализация reverse shell на Python с поддержкой кириллицы (кодировка CP866 для Windows). 

**Архитектура:**
- **Сервер (listener):** Машина атакующего. Принимает подключение и отправляет команды.
- **Клиент (victim):** Машина жертвы. Подключается к серверу и выполняет команды в cmd.exe.

**Особенности:**
- Поддержка русского языка в выводе команд (кодировка CP866)
- Переиспользование порта (SO_REUSEADDR)
- Простой shell-интерфейс

## 🛠️ Файлы

- `server.py` - серверная часть (слушает входящие подключения)
- `client.py` - клиентская часть (подключается к серверу)
